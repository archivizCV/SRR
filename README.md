# 效果图修改标准|Standards for Rendering Revisions(SRR)
## 简介 | Introduction

效果图的修改标准一直以来主要依靠肉眼以及经验来辨识，这带来了两个问题：01 标准本身不够精确，02 在沟通和判断上需要人力投入。  
本项目希望建立一套可复用的规则体系，用于对修改进行定量描述和评估，从而在一定程度上解放人力，让修改的判定更加客观、可预期。  
同时，从本质上来说，效果图修改几乎不可能被完全量化或完全规则化。因此，本项目的目标是尽可能逼近一个理想的量化模型，将那些“可以被度量”的部分标准化。

The standards for modifying renderings have traditionally relied on visual judgment and personal experience. This approach presents two main issues:  
The standards themselves lack precision.  
Communication and evaluation require manual effort.  
This project aims to establish a reusable rule system to quantitatively describe and assess revisions. The goal is to reduce human involvement to some extent and make the evaluation process more objective and predictable.  
At the same time, it is fundamentally impossible to achieve complete quantification or full standardization of rendering modifications. Therefore, the project’s objective is to approximate an ideal quantitative model as closely as possible, standardizing the aspects that can be measured.
