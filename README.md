#效果图修改标准|Standards for Rendering Revisions(SRR)
##简介|Introduction

效果图修改标准一直以来主要依靠人工与肉眼来判断修改量，既不够客观，也容易产生沟通成本和误差。
本项目旨在建立一套可执行的、可量化的规则体系，用于对效果图修改进行科学衡量与归类，从而减少人为判断带来的不确定性。

Traditionally, the workload and extent of revisions for renderings have been judged manually by eye, which is neither precise nor efficient.
Such a subjective approach often leads to ambiguous expectations, inconsistent pricing, and heavy communication overhead.
